# Healthcare SaaS Dashboard

A modern healthcare SaaS dashboard built with **React**, **TypeScript**, **TanStack Router**, **shadcn/ui**, and **TailwindCSS**.  
It provides modules for managing patients, viewing analytics, and monitoring key metrics in a clean, responsive UI.

---

## 🚀 Features

- **Dashboard**
  - Displays quick stats (patients, appointments, revenue).
  - Navigation cards to other modules.

- **Patient Management**
  - Interactive DataTable with:
    - Column filters
    - Search
    - Pagination
    - Column visibility toggles

- **Analytics**
  - Charts powered by **Recharts** with custom shadcn/ui chart components.

- **Responsive Design**
  - Works seamlessly across desktop and mobile.

---

## 🛠️ Tech Stack

- **React 18 + TypeScript**
- **Vite** (bundler)
- **TanStack Router** (routing)
- **shadcn/ui** (UI components)
- **TailwindCSS** (styling)
- **Recharts** (charts)
- **Lucide-react** (icons)
